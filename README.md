# AdditionsAPI
An API to create Custom Items with ease!

Maven Repository:
```java
<repositories>
    <!-- Additions API Repository -->
    <repository>
        <id>AdditionsAPI-mvn-repo</id>
        <url>https://raw.github.com/chrismin13/AdditionsAPI/mvn-repo/</url>
        <snapshots>
            <enabled>true</enabled>
            <updatePolicy>always</updatePolicy>
        </snapshots>
    </repository>
</repositories>

<dependencies>
    <!--Additions API-->
    <dependency>
        <groupId>com.chrismin13</groupId>
        <artifactId>AdditionsAPI</artifactId>
        <version>1.3.0-SNAPSHOT</version>
        <scope>provided</scope>
    </dependency>
</dependencies>
```
